# teacher

* 教师管理系统是一个不可缺少的重要模块。随着信息技术在管理上越来越深入而广泛的应用，管理信息系统的实施在技术上已经逐渐成熟。
* 管理信息系统是一个不断发展的新型学科，任何一个单位要生存发展，要提高效率的把内部活动有机的组织起来，就必须要建立与自身特点相适应的管理系统。
